# OIBSIP Python Programming - Task 1: Voice Assistant

## Objective
Build a Python-based voice assistant that listens to spoken commands and responds with useful actions, using speech recognition and text-to-speech.

## Tools Used
- Python
- speech_recognition
- pyttsx3
- webbrowser
- datetime

## Steps Performed
1. Captured voice input using the speech_recognition library via microphone.
2. Implemented greeting response for "hello".
3. Added functionality to tell the current time and date on request.
4. Implemented web search feature that opens a browser with the user's query.
5. Added website-opening feature for common sites (YouTube, Google, Gmail, LinkedIn, GitHub).
6. Handled unrecognized speech gracefully by prompting the user to repeat.
7. Integrated text-to-speech feedback using pyttsx3 for all responses.

## Outcome
A working command-line voice assistant that responds to greetings, tells time/date, performs web searches, and opens common websites — with graceful error handling for unclear speech.
